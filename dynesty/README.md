# Installing and Running dynesty
```
conda create -n dynesty python=3.12
conda activate dynesty
pip install dynesty
```
```
conda deactivate
```
```
python Ahmad.py
```
# requirements

```
pip install numpy scipy matplotlib
```
```
pip install pytest
```
```
pip install pytest-xdist
```
```
pip install notebook
jupyter notebook
```
