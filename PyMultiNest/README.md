# Installing and Running jaxns
```
conda create -n pymultinest python=3.12
conda activate pymultinest
pip install pymultinest
```
```
conda deactivate
```
```
python Ahmad.py
```

# requirements
```
sudo apt update
sudo apt install gfortran libblas-dev liblapack-dev
```
